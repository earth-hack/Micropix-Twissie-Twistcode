Machine Learning Library
- [Pytorch](https://pytorch.org/) 
- FastAI (https://www.fast.ai/)

## Application Setup
- Install all the library required
```
pip3 install flask
pip3 install torch torchvision
pip3 install fastai
```
- Run the script to initialize the server
```
python3 server.py
```
- Point your browser to 
`http://localhost:5000`
